# Sales Dashboard API

A Node.js + Express API to serve sales data from `sales.json` for dropdowns, date filters, and dashboard charts/cards.

Step:

1. npm install
2. npm run dev
3. A Postman collection is provided for easy testing: **`test.postman_collection.json`**
